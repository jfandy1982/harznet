# Scripts and configurations for a personal LAN setup

This repository stores the administrative scripts and configurations used in my private LAN 'HARZNET'. The several computer setups are also reflected in the folder structure of this repository. There should be sufficient documentation available in each subfolder describing the purpose of the scripts and their usage.

It is _NOT_ planned to store personal configurations (aka dotfiles) in this repository. For that, a separate repository might be used.

The folders cover the following components and setups in the LAN:

- `dsm`: everything related to Synology NAS systems - as its OS is called DSM (Disk Station Manager)
